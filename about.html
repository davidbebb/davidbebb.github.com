David Bebb

Cambridge educated, inspired by technology and motivated by finding and creating solutions. Enjoys living in a world where things work as they should. An enthusiastic science communicator, I was a key member of the team that opened Cambridge's first interactive science museum.

Relevant software design and development projects


Healthera medicine system: a combined medication reminder app for patients and compliance data collection/collation service for pharmacists/NHS.

  • Developed an extensible system to interpret medicine information from pharmacy dispense labels, a key component of the Healthera offering.
  • Designed and implemented key elements of a series of highly modular RESTful APIs to interface with mobile, desktop and embedded clients.
  • Technologies:  NodeJs, MongoDB, Cassandra, Redis, Werker, AngularJS, JQuery, Bootstrap, deployed on Amazon Web Services (EC2 containers, Route53)
  • Testing: Endpoint and unit testing using Jasmine, continuous integration tests using Werker.


Cambridge Science Centre: Raspberry Pi based exhibit content and live data manager to display multi-levelled information in response to hardware input.

  • Designed and implemented integration of hardware input with graphical client application to allow museum visitors to access content.
  • Designed user interface tool chain for staff to upload content.
  • Technologies: NodeJs


Makers Academy: Final team project, mobile-centred webapp.

  • Technologies: AngularJs, Firebase.
